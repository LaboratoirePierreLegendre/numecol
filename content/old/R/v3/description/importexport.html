What does IMPORT-EXPORT do ? Program IMPORT-EXPORT allows to import resemblance matrices, transforming them from ASCII to the SIMIL-type binary format required for them to be used by the R data analysis programs. It can also perform the opposite operation, transforming SIMIL-type binary resemblance matrices to ASCII files.
